PYTHON ESSENTIALS

Domina las colecciones de datos, dimensiones y el control de flujos.

1. Listas: Tu primera colección

Imagina que una lista es como una fila de casilleros en tu escuela. Cada casillero tiene un número pintado en la puerta (el índice) y adentro puedes guardar cosas (el valor). ¡El único truco que debes recordar es que en programación, el primer casillero siempre es el número 0!

D

Crear una Lista

Se definen usando corchetes []. Vamos a guardar las temperaturas de los primeros tres días de la semana.

# Lunes: 22°, Martes: 25°, Miércoles: 19°

temps = [22, 25, 19]

A

Leer y Modificar

Usa el número del "casillero" (índice) entre corchetes para ver qué hay adentro o cambiarlo.

# Leer la temperatura del Martes (índice 1)

martes = temps[1]

# Corregir el Lunes (índice 0) a 24°

temps[0] = 24

Observa cómo cada "casillero" (Día 0, 1, 2) guarda un valor (Temperatura).

2. Matrices: Agregando otra dimensión

Si una lista es una fila de casilleros, una matriz es como los asientos de un cine o un tablero de ajedrez. Para encontrar tu lugar, ya no basta con un solo número; necesitas saber la fila y la columna (el número de asiento). ¡Es una lista que adentro tiene más listas!

Sintaxis de Matrices

Se accede siempre primero a la Fila (hacia abajo) y luego a la Columna (hacia la derecha): matriz[fila][columna].

# Creando un tablero 2x3

tablero = [

[1, 2, 3], # Fila 0

[4, 5, 6] # Fila 1

]

# Acceder al número 6

dato = tablero[1][2]

# Cambiar el 2 por un 9

tablero[0][1] = 9

3. Ciclos: Automatizando el trabajo

Los ciclos (o bucles) son los motores de la programación. Nos permiten indicarle a Python que repita una acción muchas veces sin tener que escribir el código una y otra vez. Es la forma en la que "recorremos" o revisamos todos los datos de nuestras listas.

El Ciclo FOR

Es el ciclo definido. Úsalo cuando sepas exactamente cuántos elementos tienes que revisar (por ejemplo, recorrer todos los elementos de una lista, uno por uno).

for temperatura in temps:
    # Este bloque se repite por cada dato
    print(f"Hizo {temperatura} grados")

El Ciclo WHILE

Es el ciclo condicional. Úsalo cuando quieres que algo se repita mientras una condición siga siendo verdad. Requiere que controles manualmente el índice.

i = 0
while i < len(temps):
    print(temps[i])
    i += 1  # ¡No olvides sumar, o será infinito!

Control Especial: Break y Continue

A veces necesitas saltarte las reglas normales de los ciclos. Aquí es donde entran estos dos comandos especiales que alteran el flujo de ejecución.

CONTINUE

Ignora el resto del código en esta iteración y salta a la siguiente vuelta del ciclo inmediatamente.
