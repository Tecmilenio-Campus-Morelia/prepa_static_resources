Listas: Tu Colección Ordenada

Una lista es como una estantería numerada. Puedes guardar cualquier tipo de dato (texto, números) y encontrarlos siempre en el mismo orden. Recuerda la regla de oro: ¡Los índices siempre empiezan en 0!

Sintaxis Clave

frutas = ["manzana", "pera", "plátano"]

Acceso: frutas[0] es "manzana"

Matrices: Listas de Listas

Una matriz es una cuadrícula de datos. Imagínala como una hoja de cálculo o un tablero de ajedrez. Para acceder a un dato, necesitas dos coordenadas: [fila][columna].

matriz = [

  [1, 2, 3], # Fila 0

  [4, 5, 6], # Fila 1

  [7, 8, 9]  # Fila 2

]

# Acceder al centro (5):

centro = matriz[1][1]

Mapa de Coordenadas de la Matriz

Cada burbuja es un valor en su posición [Fila, Columna].

Moderniza tu Código: f-strings

🚫 La forma antigua (Difícil)

nombre = "Juan"
puntos = 10
print("Hola " + nombre + ", tienes " + str(puntos) + " pts.")

❌ Propenso a errores, difícil de leer.

✅ La forma Python 3.6+ (f-strings)

nombre = "Juan"
puntos = 10
print(f"Hola {nombre}, tienes {puntos} pts.")

✨ Limpio, rápido y legible.

Ciclos y Control

Los ciclos nos permiten automatizar tareas repetitivas. Recorrer una lista de 1000 alumnos toma el mismo esfuerzo que una de 3.

  • F FOR: Para cuando sabes cuántas veces repetir (recorrer listas).
  • W WHILE: Para cuando repites hasta cumplir una condición.

Análisis de la Estructura

Los gráficos mostrados arriba no son solo decorativos. El Gráfico de Barras demuestra cómo cada índice (0, 1, 2) corresponde a un valor único y específico, y cómo podemos analizar propiedades de esos datos (como la longitud del texto). El Gráfico de Dispersión (Matriz) es fundamental para entender que una matriz es espacial: necesita un eje X (Columnas) y un eje Y (Filas) para ubicar un dato, igual que en un plano cartesiano.
